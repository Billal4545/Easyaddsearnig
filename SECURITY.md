## Reporting a Vulnerability

We take the security of our application very seriously. If you discover a security vulnerability or bug, please do **NOT** open a public issue on GitHub. 

Instead, please report it privately so we can fix it before it is exploited.

### How to Report:
Please send an email to: **[mkbillal791@gmail.com]**

**Please include the following details in your email:**
* A clear description of the vulnerability.
* Step-by-step instructions to reproduce the issue.
* The potential impact (what a hacker could do with this bug).

### Response Timeline:
* We will acknowledge receipt of your vulnerability report within **48 hours**.
* We will send you regular updates about our progress in fixing the issue.
* Once the issue is fixed, we will notify you before public disclosure.

Thank you for helping keep our project safe!
